# Market Correlation Radar 📊

**Market Correlation Radar** は、異なる金融資産（株式、為替、債券、コモディティ、暗号資産）間の「相関関係」を可視化し、ポートフォリオのリスク管理や投資戦略の立案を支援する Streamlit アプリケーションです。

単なる価格推移の比較ではなく、**日次変化率（Daily Returns）** に基づく純粋な相関係数を計算し、AIアナリストが自動的にポートフォリオの診断（集中リスク、偽のヘッジ、有望な分散先など）を行います。


## ✨ 主な機能

1.  **相関ヒートマップ (Correlation Matrix)**
    - 選択した銘柄間の相関係数（-1.0 ～ 1.0）をヒートマップで可視化。
    - 「何と何が連動しているか」「どのペアが逆相関（ヘッジ）か」が一目で分かります。

2.  **AI Analyst Insights (投資戦略診断)** 🧠
    - 相関行列を自動分析し、以下のインサイトを提供します：
        - **⚠️ 集中リスク警告**: 相関が非常に高い（0.7以上）ペアを警告。
        - **🚨 ヘッジ機能不全 (Fake Hedge)**: 「守り資産（金・債券）」と「攻め資産（ハイテク・クリプト）」が異常に連動している危険な状態（0.5以上）を検知。
        - **🛡️ ヘッジ機能**: ポートフォリオのリスク分散に寄与している逆相関ペアを発見。
        - **🧘 独立独歩**: 他の資産と連動しない、分散効果の高い銘柄を特定。

3.  **隠れトレンド銘柄の自動発掘** 🚀
    - 米国 Yahoo Finance の「Most Active」データをリアルタイムで取得。
    - **独自アルゴリズム**:
        - 単なる出来高だけでなく「売買代金 (Trading Value)」で重み付けし、ペニー株を除外。
        - 既にリストにある超大型株（NVDA, TSLAなど）は除外し、**「今、資金が集中している中型・新興トレンド銘柄」** だけをピックアップします。

4.  **相対パフォーマンスチャート**
    - 期間の初日を基準（0%）とした累積リターンを表示し、パフォーマンスの優劣を比較できます。

5.  **設定の保存**
    - 選択した銘柄や期間は自動的に保存され、次回起動時に復元されます。
  
## 📦 使用技術

- **[Streamlit](https://streamlit.io/)**: フロントエンド・UIフレームワーク
- **[yfinance](https://github.com/ranaroussi/yfinance)**: 株価データの取得
- **Pauldas & NumPy**: データ解析・計算
- **Seaborn & Matplotlib**: ヒートマップ描画
- **Requests & lxml**: Webスクレイピング（トレンド銘柄取得）
